# แปลง json 2 มิติเป็น csv, csv กลับเป็น json 2 มิติ

## Requirement

- NodeJS

## Getting Started

```bash
npm install

```

## Convert

เอาไฟล์มาวางในโปรเจ็ค
ชื่อไฟล์ label.json

```bash
node convert.js
```

จะได้ไฟล์ output.csv

## Revert

เอาไฟล์มาวางในโปรเจ็ค
ชื่อไฟล์ output.csv

```bash
node revert.js
```

จะได้ไฟล์ output.json
